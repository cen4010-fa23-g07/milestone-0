About me:

I'm a computer science student with a keen interest in compiler design, functional programming, type theory, category theory, and computer science theory in general. I've always been fascinated by the inner workings of programming languages and how they're compiled into machine code, and I'm always eager to learn about new techniques and tools to improve code efficiency and performance. When I'm not studying, you can often find me reading programming and theory books, exploring new coding libraries, and analyzing chess strategies and positional plays.

I enjoy applying the same critical thinking skills that I use in programming to analyzing chess games, and I find that it helps me to think more strategically in both areas. I also occasionally enjoy cycling on my free time.
